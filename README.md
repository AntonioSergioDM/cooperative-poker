# ♥️ ♣️ Cooperative Poker ♦️ ♠️

## How to use
Clone the repo

Install it and run:

```bash
npm install
npm run dev
```

Now you can play Cooperative Poker locally ♥️♣️♦️♠️

## Docker
You can run on docker by creating the .env file and running:
```bash
docker compose up -d --build
```

## Deploy
- Navigate to the directory where the Docker Compose file is located
```bash
cd .. && cd .. && cd /mnt/user/Storage/ml-automation  
```
- Run docker compose to build and start the containers
```bash
docker compose up -d --build
```

## The idea behind the example
We wanted to play a nice Cooperative Poker game online.

We used React, Next and Socket.IO.
